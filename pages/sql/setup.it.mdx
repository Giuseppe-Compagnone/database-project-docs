# Setup

Iniziamo ora ad implementare il database lato codice, lavorando in un ambiente MySQL:

## Creazione del Database

```sql
CREATE DATABASE elearning_system;
```

Successivamente lo selezioniamo per essere sicuri di lavorare su di esso:

```sql
USE elearning_system;
```

Ora siamo pronti per strutturare il database secondo le specifiche scritte precedentemente.
