# Stack di tecnologie

Ora definiamo la struttura di un'interfaccia web che permetta agli utenti di interagire con il database. Puoi accedere all'interfaccia <a className="cta" href="https://elearning-system.netlify.app/" target="_blank"> qui</a>. Innanzitutto, definiamo lo stack di tecnologie che utilizzeremo:

## Frontend

### React

React è una libreria JavaScript per la creazione di interfacce utente (UI). Viene utilizzato per costruire componenti riutilizzabili che compongono l'interfaccia grafica di un'applicazione web.

### SCSS

SCSS (Sassy CSS) è un'estensione di CSS che offre funzionalità avanzate come variabili, nesting e mixin. Viene utilizzato per scrivere fogli di stile CSS più organizzati e mantenibili.

### Webpack

Webpack è un bundler JavaScript che viene utilizzato per gestire le dipendenze delle risorse web (come JavaScript, CSS e immagini) e per creare bundle ottimizzati per la produzione. È spesso utilizzato per il processo di build di progetti frontend, inclusa la gestione dei moduli e la generazione di asset.

## Backend

### Node.js

Node.js è un ambiente di runtime JavaScript lato server che consente di eseguire codice JavaScript sul server. Viene utilizzato per la creazione di applicazioni web scalabili e ad alte prestazioni.

### Express

Express è un framework web per Node.js che semplifica la creazione di API RESTful e applicazioni web. Fornisce una serie di funzionalità utili per la gestione delle richieste HTTP, il routing, la gestione delle risposte e altro ancora.

## Database

### MySQL

MySQL è un sistema di gestione di database relazionale open-source ampiamente utilizzato. Viene utilizzato per archiviare e gestire i dati in modo strutturato, consentendo operazioni di lettura e scrittura efficienti.

### Linguaggio SQL

Il linguaggio SQL (Structured Query Language) è un linguaggio di programmazione utilizzato per comunicare con i database relazionali. Viene utilizzato per definire, manipolare e recuperare dati all'interno di un database MySQL.
