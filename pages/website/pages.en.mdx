# Pages

The website will have the following pages:

- Home:
  The main page of the site containing a welcome message.

- SignUp:
  The page used for signing up to the site.

- Login:
  The page used for logging into the site.

- Performances:
  On this page, students can view their own performances.

- Courses:
  On this page, students can view the courses they are enrolled in, and teachers can view the courses they have created. Additionally, teachers can create new courses from this page.

- Course:
  This page contains information about a specific course and the course materials. Teachers can also upload materials and record student performances for that course from this page.

All pages are accessible only after authentication, except for SignUp and Login.
